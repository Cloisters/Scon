About Scon

What is Scon?

Scon is a powerful service enumeration tool that allows you to quickly and easily identify all the services running on a particular target network or system.

Whether you're a security professional looking to assess the security posture of a network, or a network administrator trying to get a better understanding of what's running on your systems, Scon is the tool for you.

Features

  • Fast and reliable service enumeration
  • Support for a wide variety of protocols and services
  • Easy-to-use command-line interface
  • Customizable output formats
  • Integration with other security tools

How to Use Scon

To use Scon, simply download and install the tool on your system. Once installed, you can run Scon from the command line using the following syntax:

git clone https://github.com/Cloisters/Scon
cd Scon
chmod +x Scon
./Scon

For more information on how to use Scon, including a list of all available options and examples of common usage scenarios, please refer to the Scon documentation.
